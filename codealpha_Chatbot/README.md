# 🤖💬 Chatbot Project 💬🤖
📖 Description
A simple Python Chatbot project that interacts with users through predefined responses. The chatbot accepts user input, processes queries, and provides appropriate replies, making it a beginner-friendly conversational AI application.
## ✨ Features
✅ User-friendly console interaction
✅ Responds to predefined user queries
✅ Continuous conversation using loops
✅ Input validation
✅ Simple and easy-to-understand logic
✅ Beginner-friendly AI project
## 🛠️ Concepts Used
🔹 Python
🔹 Strings
🔹 Dictionaries
🔹 If-Else Statements
🔹 While Loop
🔹 User Input Handling
## 📊 Example
📌 User: Hello
📌 Chatbot: Hi! How can I help you?
📌 User: How are you?
📌 Chatbot: I am doing great. Thank you!
