Hangman Game
Hangman is a simple and fun word-guessing game played by two or more players. One player thinks of a secret word, and the other player(s) try to guess it by suggesting letters. �
Hangman Words +1
Rules:
One player chooses a secret word.
Draw a blank line (_) for each letter in the word.
Example: For the word APPLE, draw:

_ _ _ _ _
The guessing player says one letter at a time.
If the letter is in the word, write it in all correct positions.
Example:

A _ _ _ _
If the letter is not in the word, draw one part of the hangman figure.
The game continues until:
The word is guessed correctly (Win), or
The hangman drawing is completed (Lose). �
