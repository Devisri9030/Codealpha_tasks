# 📈💰 Stock Portfolio Tracker 💰📈
📖 Description
A simple Python project that calculates total stock investment using user input and predefined stock prices.

## ✨ Features
✅ User enters stock name and quantity

✅ Uses dictionary for stock prices

✅ Calculates total investment

✅ Displays result

✅ Saves portfolio data to a CSV file

## 🛠️ Concepts Used
🔹 Dictionary

🔹 Input/Output

🔹 Arithmetic Operations

🔹 File Handling (CSV)

## 📊 Example
📌 AAPL → 10 Shares → ₹1800 Total

🎯 Project Highlights
🌟 Easy-to-use console application

🌟 Real-time investment calculation

🌟 CSV file generation

🌟 Beginner-friendly Python project

🌟 Demonstrates practical use of dictionaries and file handling
